Timing and Buffer Control

MPEG passes the source timing to the decoder by means of time stamps.  This is explained fully under Timing Model and the other subtopics of this part.

MPEG does not send as part of the data a specific current value of buffer fullness.  It does, however, include redundant equivalent information. This is explained under the subtopics of this part:

  • Timing Model

  • Timing Model Diagram

  • PCR - Program Clock Reference

  • STC - System Time Clock

  • SCR - System Clock Reference

  • PTS - Presentation Time Stamp

  • DTS - Decode Time Stamp

  • Multiplexed Programs

  • VBV - Video Buffering Verifier

  • Buffer Synchronization and Startup

  • Low Delay Mode

  • Decoder STC Synchronization
