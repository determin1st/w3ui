Files installed

This tutorial help file, MPEG2TUT.HLP, was developed using Hypertext Studio Professional Editon. (Olson Software, www.olsonsoft.com)

 

Setup installs the MPEG2TUT.HLP file and the MPEG2TUT.CNT file to the user specified directory and also installs a library file, OSHTL332.DLL, in the windows/system folder, as well as an uninstall file.  The .DLL library file is needed for operation of certain features, particularly the alphabetical glossary index.
