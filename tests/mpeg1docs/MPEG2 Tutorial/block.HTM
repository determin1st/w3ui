Block

A block is an 8 x 8 array of pixels. Four luma blocks plus a number of chroma blocks that depends on the chroma pixel structure (see Pixel Positioning ) indicated in the sequence header comprise a macroblock.

A block is the fundamental unit for the DCT (discrete cosine transform).
