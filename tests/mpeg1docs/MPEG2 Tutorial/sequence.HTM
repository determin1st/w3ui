Sequence

A Sequence begins with a unique 32 bit start code followed by a header.

The header carries:

  • picture size

  • aspect ratio

  • frame rate and bit rate

  • optional quantizer matrices

  • required decoder buffer size

  • chroma pixel structure

  • optional user data

 

The sequence information is needed for channel changing. The sequence length depends on acceptable channel change delay.
